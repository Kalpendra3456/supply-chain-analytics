# 📦 Supply Chain Analytics Dashboard

## 📌 Project Overview

The **Supply Chain Analytics Dashboard** is an end-to-end data analytics project that analyzes supply chain operations to help businesses improve efficiency, reduce costs, and make data-driven decisions.

The project covers the complete analytics workflow, including data generation, data cleaning, SQL analysis, and interactive Power BI dashboard development.

---

# 🎯 Objectives

* Analyze supply chain performance.
* Monitor inventory levels.
* Track orders and deliveries.
* Evaluate supplier performance.
* Identify delayed shipments.
* Improve operational efficiency through data-driven insights.

---

# 🛠️ Tools & Technologies

* **SQL** – Data analysis and querying
* **Python (Pandas, Faker)** – Data generation and preprocessing
* **Excel** – Data validation
* **Power BI** – Dashboard and visualization
* **Power Query** – Data transformation
* **DAX** – KPI calculations
* **Git & GitHub** – Version control
